# Employee Management System

## Project Description
The Employee Management System is a Java-based desktop application developed using Object-Oriented Programming concepts and MySQL database integration. The system helps organizations manage employee records, departments, attendance, salaries, and leave management efficiently.

---

## Technologies Used
- Java
- Java Swing
- MySQL
- JDBC
- NetBeans IDE

---

## OOP Concepts Used
- Classes and Objects
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Exception Handling
- Packages
- JDBC Connectivity

---

## Main Modules
1. Login System
2. Employee Management
3. Department Management
4. Attendance Management
5. Salary Management
6. Leave Management
7. Reports Dashboard

---

## Database
Database Name: ems_db

Import the `database.sql` file into MySQL before running the project.

---

## How to Run the Project

1. Open the project in NetBeans IDE
2. Add MySQL Connector JAR file
3. Import database.sql into MySQL
4. Run LoginForm.java
5. Login using:
